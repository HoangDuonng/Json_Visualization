## Custom Shortcuts

You can customize shortcuts in Settings:

1. Click **Settings** → **Keyboard Shortcuts**
2. Find action to customize
3. Click shortcut field
4. Press new key combination
5. Click **Save**

### Reset Shortcuts

1. Click **Settings** → **Keyboard Shortcuts**
2. Click **Reset to Defaults**
3. Confirm reset

## Tips

**Learn gradually:**
- Start with common shortcuts (Ctrl+S, Ctrl+F)
- Add more as you get comfortable
- Focus on your most-used actions

**Practice:**
- Use shortcuts instead of mouse
- Create muscle memory
- Speed increases over time

**Customize:**
- Change shortcuts to match your workflow
- Use familiar shortcuts from other tools
- Keep frequently-used actions accessible

## Accessibility

### Keyboard-Only Navigation

Full keyboard support for accessibility:

1. Enable **Keyboard Navigation** in Settings
2. Use Tab to move between elements
3. Use Arrow keys within components
4. Use Enter/Space to activate
5. Use Esc to cancel/close

### Screen Reader Support

Optimized shortcuts for screen readers:

- All actions have keyboard shortcuts
- Focus indicators visible
- ARIA labels on all controls
- Announcements for state changes
